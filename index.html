저는 섬세하고 집요한,
직관적인 UI와
기억에 남는 UX를 만드는
프런트엔드 엔지니어입니다.

디자인 전공을 접목시켜 프런트엔드 개발 역량을 탄탄히 쌓아나갔으며,
웹 표준과 접근성을 준수하며 12%의 브라우저 성능 향상을 이끈 경험이 있습니다.
또한, 백엔드 개발 역량 또한 빠트리지 않고 습득하여
홀로 풀스택 프로젝트를 배포까지 진행한 경험도 있습니다.

저는 이런 프로젝트를 경험했어요.

  • Palette

    AI 기반 키워드별 컬러 추천 및 피킹 서비스
    w/ Next.js, Typescript, TailwindCSS, Openai

  • Galaxy Market

    C2C 중고 거래 서비스 플랫폼
    w/ Next.js Typescript, SWR, TailwindCSS, Prisma

  • At Flix

    영화 & TV 시리즈 탐색 서비스
    w/ Typescript, React, styled-component, react-query, framer-motion

  • Rello

    스케줄 칸반 보드 서비스
    w/ Typescript, React, styled-components, recoil

  • Coin Track

    암호 화폐 트래킹 서비스
    w/ Typescript, React, recoil, ApexCharts

  • Music Tree

    음원 & 아티스트 탐색 서비스
    w/ Javascript, React

저는 경험에서 배운 것을 기록으로 남기고 있어요.

  1. 포트폴리오 웹사이트 회고
  2. JSON 전역 객체의 메서드 직접 구현하기
  3. HTTP Keep Alive와 WebSocket의 차이
  4. Typescript 개발 환경 설정하기
  5. Next.js 클라이언트 사이드 환경 변수 설정하기
  6. 화살표 함수의 제네릭 in .tsx

저는 이런 기술을 사용할 수 있어요.

  1. HTML5, CSS3(SCSS), Javascript(ES6+), Typescript
  2. React, Next.js, Recoil, Redux
  3. Jest, React Testing Library
  4. Tailwind, Styled-component
  5. Node.js(w/ Express.js), Prisma, MongoDB(w/ Mongoose)
  6. Git, Github
  7. Adobe Creative suites, MS Offices
  8. Mac OS, Windows

저는 이런 경험도 있어요.

  1. 개발이 있어 필수적인 문제 해결 능력 향상 및 로직 개선을 위해 500개 이상의
    알고리즘 문제를 꾸준히 풀며, Github와 블로그에 풀이 과정을 정리하고 있어요.
  2. 프런트엔드 개발 역량을 더욱 향상시키기 위해 인접된 분야의 웹디자인기능사,
    컴퓨터활용능력1급, GTQ그래픽기술자격1급 등 자격 시험에 응시해 자격을 취득했어요.

이럴 때 제가 도움이 될 수 있을거에요 👍

  1. 1px의 오차도 허용하지 않는 섬세함과 집요함이 필요할 때
  2. HTTP보다는 WebSocket처럼 커뮤니케이션하는 동료가 필요할 때
  3. 마르지 않는 동기부여로 팀의 엔진이 되는 엔지니어가 필요할 때

    4. 함께 일할 프런트엔드 엔지니어를 찾고 계신가요?
